Select

A select component that allows users to select from a list of options.

Installation

Install the component
pnpm add @nipsys/shadcn-lsd

Import

Import the component
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectItem,
  SelectLabel,
  SelectSeparator,
  SelectTrigger,
  SelectValue,
} from '@nipsys/shadcn-lsd';

Usage

Basic select
<Select>
  <SelectTrigger className="w-[180px]">
    <SelectValue placeholder="Select a framework" />
  </SelectTrigger>
  <SelectContent>
    <SelectGroup>
      <SelectLabel>Frameworks</SelectLabel>
      <SelectItem value="react">React</SelectItem>
      <SelectItem value="vue">Vue</SelectItem>
      <SelectItem value="angular">Angular</SelectItem>
    </SelectGroup>
  </SelectContent>
</Select>

Basic Usage

A basic select with a placeholder and a list of options. Use defaultValue to set an initial value.

API Reference

Select Props

valuestring
Controlled value of the select.
Default: undefined
defaultValuestring
Initial value of the select (uncontrolled).
Default: undefined
onValueChange(value: string) => void
Callback when the selected value changes.
Default: undefined
disabledboolean
Disables the select.
Default: false
namestring
Name of the select (for form submission).
Default: undefined
requiredboolean
Marks the select as required.
Default: false

SelectTrigger Props

size"default" | "sm"
Size of the select trigger.
Default: "default"
disabledboolean
Disables the select trigger.
Default: false

SelectItem Props

valuestring
Required. The value of the item.
disabledboolean
Disables the item.
Default: false

Accessibility

Selects follow WAI-ARIA guidelines and are fully accessible. The component uses proper ARIA roles and attributes to ensure screen readers can properly announce the select, its options, and the current selection. Keyboard navigation is fully supported with arrow keys, Enter, and Escape.
